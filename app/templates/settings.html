Navidrome / Subsonic

Used by Auto Profile (now playing) and Auto Toots (album finished, loved tracks, weekly top artists).

Spotify

Used by Auto Profile to show your currently playing track in your Mastodon bio.

Create an app at developer.spotify.com, then get a refresh token with user-read-currently-playing user-read-recently-played scopes. One-time auth guide →

Jellyfin

Used by Auto Profile to show what you're currently watching in your Mastodon bio.

Get an API key from Dashboard → API Keys. User ID is optional — leave blank to show any active session.

Plex

Used by Auto Profile to show what you're currently watching in your Mastodon bio.

Find your token at plex.tv/claim or by inspecting any Plex API request (look for X-Plex-Token).

Tautulli

Alternative to Plex for now-watching updates in your Mastodon bio.

Get an API key from Settings → Web Interface → API. Alternative to using Plex directly.

Letterboxd

Goodreads
